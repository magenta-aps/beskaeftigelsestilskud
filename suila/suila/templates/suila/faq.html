    {% blocktranslate trimmed %} Få svar på de mest sandsynlige årsager til, at du ikke har fået udbetalt Suila-tapit. {% endblocktranslate %}

    {% blocktranslate trimmed %} Suila-tapit udbetales, når Skattestyrelsen modtager signaler fra arbejdsmarkedet. De fire signaler er: {% endblocktranslate %}

    1. {% translate "A-indkomst fra lønarbejde, som arbejdsgiverne indberetter i Sulinal." %}
    2. {% translate "A-indkomst fra indhandling, som indhandlingsstederne indberetter i Sulinal." %}
    3. {% translate "Betalte B-skatter. Hvis du har forskudsregistreret B-indkomst, vil du modtage en opkrævning af B-skatter. Når den betales, modtages signal." %}
    4. {% translate "Indberetning af udbytte. Når selskaberne indberetter udbytte, giver dette signal om en indkomst." %}

    {% blocktranslate trimmed %} Ud fra signalerne om indkomst beregnes en forventet årsindkomst. Suila-tapit udbetales til borgere med en forventet årsindkomst på 70.000 kr. – 500.000 kr. Dog vil borgere med en forventet årsindkomst på 475.000 - 500.000 kr. først modtage Suila-tapit, når Skattestyrelsen kan afklare, om den forventede årsindkomst holder sig under 500.000 kr. {% endblocktranslate %}

    {% blocktranslate trimmed %} Bemærk, at der går to måneder fra den måned, din indkomst vedrører, til din udbetaling af Suila-tapit. Fx udbetales Suila-tapit for januar måned i marts, for februar måned i april, osv. {% endblocktranslate %}

    {% translate "Hvis du ikke har modtaget Suila-tapit, kan der være følgende forklaringer:" %}

    {% url "suila:person_detail_income" pk=person.pk as person_income_url %} {% blocktranslate trimmed with url=person_income_url %} Din arbejdsgiver eller dit indhandlingssted har ikke indberettet din indkomst. Eller har indberettet for sent eller forkert. Måske har de slet ikke indberettet. På siden "Mine signaler" kan du se, hvilke oplysninger Skattestyrelsen har om dine indkomster. Hvis der mangler signal om indkomst fra arbejde og indhandling, så kontakt din arbejdsgiver eller dit indhandlingssted. {% endblocktranslate %}

    {% blocktranslate trimmed %} Du er selvstændig erhvervsdrivende og har ikke indbetalt løbende B-skatter. Eller du har betalt et beløb, der er så stort, at Skattestyrelsens beregning viser, at du forventes at komme til at tjene over 500.000 kr. om året. Med forskudsregistrering på Sullissivik.gl kan du give os oplysninger om dine indkomster som selvstændig erhvervsdrivende. Ud fra din forskudsregistrering sender Skattestyrelsen månedlig opkrævning af B-skatter. Når du betaler dem, giver du os signal om din indkomst. {% endblocktranslate %}

    {% blocktranslate trimmed with this_year=this_year last_year=this_year|add:-1 %} Hvis Skattestyrelsens oplysninger fra {{ last_year }} viser, at din årsindkomst nærmede sig 500.000 kr., og din månedlige indkomst varierede meget i løbet af året, vil du ikke modtage Suila-tapit månedligt i {{ this_year }}. {% endblocktranslate %}

    {% blocktranslate trimmed with this_year=this_year next_year=this_year|add:1 %} Hvis Skattestyrelsen i løbet af {{ this_year }} modtager oplysninger, der viser, at din forventede årsindkomst er under 500.000 kr., og der samtidig kommer et signal fra arbejdsmarkedet, vil Skattestyrelsen igangsætte månedlige udbetalinger fra januar {{ next_year }}. {% endblocktranslate %}

    {% blocktranslate trimmed %} Hvis Skattestyrelsen i løbet af året modtager oplysninger, der viser, at din forventede årsindkomst er under 70.000 kr. eller over 500.000 kr., er du ikke berettiget til Suila-tapit. {% endblocktranslate %}

    {% blocktranslate trimmed %} Når Skattestyrelsen har modtaget oplysninger om din samlede årsindkomst, vil du få udbetalt eventuelt manglende Suila-tapit. {% endblocktranslate %}

    {% blocktranslate trimmed %} Suila-tapit er et beskæftigelsesfradrag, der udbetales til borgere med indkomster i form af lønindkomst, indhandlingsbeløb, overskud ved selvstændig virksomhed samt modtaget udbyttebetaling. Hvis dine indkomster stammer fra fx uddannelsesstøtte, alderspension eller sociale ydelser, er du ikke berettiget til Suila-tapit. {% endblocktranslate %}

    {% blocktranslate trimmed %} Det er kun beløb på 100 kr. eller mere, der udbetales månedligt. Hvis det samlede beløb, du skal have udbetalt, er mindre end 100 kr., vil udbetalingen blive overført til næste måned. {% endblocktranslate %}
